# 2 Retired Guys Lawn Care — Website

A single-page website for 2 Retired Guys Lawn Care. Plain HTML — no build step, no dependencies. Just static files.

## Files
- `index.html` — the whole website
- `assets/` — logo and lawn photos
- `.nojekyll` — tells GitHub Pages to serve the files as-is

## Put it online with GitHub Pages (free)

1. Create a free account at https://github.com if you don't have one.
2. Click the **+** (top right) → **New repository**. Name it (e.g. `2-retired-guys`), set it **Public**, click **Create repository**.
3. On the new repo page, click **uploading an existing file**.
4. Drag in **everything from this `deploy` folder** — `index.html`, the `assets` folder, and `.nojekyll` — then click **Commit changes**.
5. Go to **Settings** → **Pages** (left sidebar).
6. Under **Branch**, pick **main** and **/ (root)**, click **Save**.
7. Wait about a minute. Your site will be live at:
   `https://YOUR-USERNAME.github.io/2-retired-guys/`

## Editing later
Open `index.html` in any text editor. Phone numbers, service text, and the Facebook link are all plain text you can search for and change. To swap a photo, drop a new image into `assets/` and update the matching file name in `index.html`.
